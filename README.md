# bct-ia-noteapp
BCT IA

# Prereqs
- Download Ganache - https://trufflesuite.com/ganache/
- Install node - https://nodejs.org/en
  
# Installation and Set up
- npm install
- cd NoteApp_Flask
- python3 -m venv venv
- venv\Scripts\activate
- pip install -r requirements.txt

# Run
- flask --app notes_app run
